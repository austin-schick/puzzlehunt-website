CMU 15-112 Spring 2018: Fundamentals of Programming and Computer Science
Check 4 (Due Tuesday 6-Feb, at 10am)

  • This check should take 2-3 hours to complete.

  • Checks can be completed collaboratively or individually. If you work collaboratively, keep group sizes reasonable and list your teammates in the writeup.

  • To start:
    1. Create a folder named 'week4'
    2. Download check4.txt to that folder
    3. Edit check4.txt in a text editor of your choice
    4. When you are ready, submit your edited check4.txt to Autolab.
  1. Week3 Time Report [25pts]
    First fill out this week3 time report. We very much need and value your thoughtful and accurate feedback in our quest to continue improving the course.

  2. Week4 Preparation Questions [75pts]
    Watch the videos and read through the notes on the following pages, answering the questions in check4.txt as you go:
    • 1D Lists and Tuples (Sections 1-17)
    • Graphics (Sections 1-8)
    Don't rush through the videos at the expense of learning! Try copying and running the code yourself as you watch. Edit the code examples slightly and predict what will happen, then run the edited code to confirm your guess. Learning is not a spectator sport; you have to keep your head in the game if you want to keep up.
